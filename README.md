# 🏥 HOSPITAL EMERGENCY ROOM DASHBOARD

---

# 📌 OVERVIEW

The **Hospital Emergency Room Dashboard** is a comprehensive data analytics project developed using **Power BI** to monitor and analyze hospital emergency room operations.

This dashboard converts raw healthcare data into interactive visual reports that help hospitals understand patient flow, emergency room performance, waiting times, and admission trends.

The project was created with the objective of improving healthcare decision-making through data visualization and business intelligence techniques.

---

# 🎯 PROJECT OBJECTIVE

The main objective of this project is to analyze emergency room data and provide meaningful insights that help hospitals improve operational efficiency and patient management.

### Objectives Include:

* Monitoring emergency room patient visits
* Tracking patient waiting times
* Analyzing admission and discharge trends
* Understanding department referrals
* Identifying peak visit periods
* Improving hospital resource planning

# 📂 DATASET INFORMATION

The dataset used in this project contains hospital emergency room records including:

* Patient ID
* Gender
* Age
* Visit Date
* Waiting Time
* Admission Status
* Department Referral
* Patient Category

The data was cleaned and transformed using **Power Query** before creating visualizations in Power BI.
