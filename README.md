🏥 HOSPITAL EMERGENCY ROOM DASHBOARD
📌 PROJECT OVERVIEW
This project is a Hospital Emergency Room Dashboard developed using Power BI to analyze and monitor emergency room operations effectively.

The dashboard provides meaningful insights into:
Patient visits
Wait times
Admission status
Department referrals
Emergency room performance
This project helps hospitals improve operational efficiency and supports better decision-making through data visualization.

📊 FEATURES

✔ Total Patient Count Analysis
✔ Average Wait Time Tracking
✔ Admission vs Non-Admission Analysis
✔ Department-wise Patient Referrals
✔ Gender Distribution Analysis
✔ Age Group Analysis
✔ Daily & Monthly Trends
✔ Interactive Filters & Visualizations

🛠️ TOOLS & TECHNOLOGIES USED
Power BI
Microsoft Excel / CSV
DAX
Data Cleaning
Data Visualization
Dashboard Design

📈 DASHBOARD INSIGHTS
Identified peak emergency room visit timings
Analyzed patient admission patterns
Tracked emergency wait time performance
Compared patient demographics
Improved hospital workflow understanding

🚀 PROJECT OUTCOME
This dashboard helps healthcare management to:
Improve emergency room efficiency
Enhance patient experience
Monitor hospital KPIs
Make faster and smarter decisions
